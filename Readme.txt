# 🚗 Vehicle Insurance Management System (MERN Stack)

This is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application for managing vehicle insurance plans and customer applications.  
It includes features for Admins, Agents, and Users.

---

## 🔧 Tech Stack

- **Frontend**: React.js + Bootstrap 5
- **Backend**: Node.js + Express.js
- **Database**: MongoDB (Atlas or Local)
- **Other Tools**: 
  - JWT Authentication
  - Cookies for session
  - Cloudinary (optional for image uploads)
  - Multer (for file upload)

---

## 💻 Prerequisites

Before running the project, make sure you have installed:

1. **Node.js** (v16 or above): [Download Here](https://nodejs.org)
2. **MongoDB** (Local or [MongoDB Atlas](https://www.mongodb.com/cloud/atlas))
3. **VS Code** (or any preferred IDE): [Download Here](https://code.visualstudio.com/)
4. **Postman** (for API testing): *Optional*
